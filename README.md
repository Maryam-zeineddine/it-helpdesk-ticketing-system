# IT Help Desk & Ticketing Management System

Full stack web app for employees to submit IT support tickets, 
and agents/admins to manage, assign, and resolve them.

## Tech Stack
- Frontend: React (Vite)
- Backend: PHP Laravel
- Database: MySQL
